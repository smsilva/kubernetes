# SEC-003 — Imagem de container sem digest fixo na app de exemplo

| Campo | Valor |
|---|---|
| **Severidade** | Baixo |
| **Script** | `scripts/08-deploy-sample-app`, linha 35 |
| **Status** | Aberto |

## Problema

A imagem do `httpbin` é referenciada sem digest imutável:

```yaml
image: kennethreitz/httpbin
```

## Impacto

Em um ambiente real, a tag `latest` implícita pode ser substituída por uma versão diferente entre execuções, quebrando a reprodutibilidade e introduzindo riscos de supply chain.

## Correção

Fixar a imagem com digest SHA256:

```yaml
image: kennethreitz/httpbin@sha256:<digest>
```

Para um lab, alternativa aceitável é usar uma imagem mantida ativamente, como `docker.io/mccutchen/go-httpbin`.
